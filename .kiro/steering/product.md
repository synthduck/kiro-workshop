# Product Overview

This is a simple e-commerce website demonstrating a complete online shopping experience with a 3-tier architecture.

## Core Features

- **Product Catalog**: Browse 20 sample products across multiple categories (Electronics, Clothing, Home, Books, etc.)
- **Product Details**: View detailed product information with customer reviews and ratings
- **Shopping Cart**: Add items, update quantities, remove items, and view total cost
- **Visual Design**: Uses emojis as product images for a fun, engaging interface

## Business Logic

- Products are organized by categories (Electronics, Clothing, Home, Books, Music, Sports, Beauty, Accessories)
- Each product has: name, price, description, emoji icon, and category
- Shopping cart supports quantity updates and automatic total calculation
- Customer reviews include ratings (1-5 stars) and text comments
- Cart items persist in the database and can be managed across sessions

## Target Audience

This is a demonstration/learning project showcasing full-stack e-commerce functionality with modern web technologies.
